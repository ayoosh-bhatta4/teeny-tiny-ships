# Organizer

Hey there! this should be the teeniest of the teeny tiny ships. And that's because it's the first one.

All this script does is take a list like :

* swiggy 350
* coffee 120
* uber 68

And converts it into a more general one:

* Food 470
* travel 120

This is just supposed to make future data processing as useful as possible.

Now you'll notice that there are several files here.

1. The first and the easiest to make is just making an api call to groq, then testing the output. The prompt and tests were designed here. 
2. The second method is making a custom model for this task.
